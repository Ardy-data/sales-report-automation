# Automated Sales Report Generator

This project automatically generates a professional Excel sales report
from a CSV file using Python and pandas.

## Features
- Calculates total and average sales
- Identifies top-performing products
- Generates a multi-sheet Excel report
- Includes an executive summary in plain English

## How to Use

1. Install Python 3.10+
2. Install dependencies:
   pip install pandas openpyxl
3. Replace `python_sales.csv` with your own sales data
4. Run:
   python sales.py
5. Open `sales_report.xlsx`

## Use Case
Small businesses and sales teams who want quick insights
without manual Excel work.
